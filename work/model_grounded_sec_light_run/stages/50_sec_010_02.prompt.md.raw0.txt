## S/E/C 评估

**Severity S → S1**
- 近光灯卡滞点亮对前车驾驶员后视镜造成的是"轻微干扰"，不是远光灯式的强眩目
- 潜在后果是前车变道/避险决策受影响，可能导致低速追尾或侧面刮擦
- 车速仅 30-60 km/h 且处于拥堵状态，碰撞能量有限，伤害程度为轻度至中度（擦伤、挫伤、轻微骨折）

**Exposure E → E3**
- 夜间城市主干路是常见驾驶场景，符合 E3 定义（4-40 小时/年，月度出现）
- 拥堵跟车在城市夜间驾驶中频繁发生

**Controllability C → C1**
- 驾驶员可感知后视镜光线异常（来源：视觉直接感知）
- 拥堵路况下车速低、车间距近，反应时间充足（>5秒）
- 驾驶员可通过调整坐姿、手动调节后视镜角度、开启防眩目功能等常规操作避免干扰
- 周围交通参与者因车速低且有空间缓冲，可通过正常减速或避让避免碰撞
